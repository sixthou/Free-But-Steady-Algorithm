# 🐰Free But Steady🐢

## 💻 자유롭지만 꾸준한 알고리즘 공부!

- 알고리즘 문제풀이 흔적을 남기자!
- 문제 선정은 개인의 자유!
- 알아서 풀고 흔적만 공유하자!
- 오직 온라인으로!
- 일주일에 3문제만 풀면 🙆‍♂️!

## 👊 참여방법

1. 저장소를 `fork` 한다
2. 본인의 github 아이디로 폴더를 생성한다.
3. 본인의 폴더에 `template/Report.md`를 복사한다.
4. 양식에 맞춰 풀이한 내용을 작성한다.
5. 커밋 규칙에 맞춰 `commit` 후 자신의 저장소에 `push`
6. 원본 저장소에 `Pull Request` 한다.
7. 매주 수요일 전까지 PR을 요청한다.
